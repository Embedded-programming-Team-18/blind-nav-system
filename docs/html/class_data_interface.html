Detailed Description

A data interface class that implements the new newScanAvail(...) of the Lidar driver.

DESCRIPTION

This class represents the data interface that is required to retrieve data from the Lidar Driver.

This class accesses the data read by the lidar after a full scan and sends pwm to the appropriate gpio pin
