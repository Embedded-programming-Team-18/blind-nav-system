Detailed Description

The LiDAR class is a low-level driver for obtaining distance values needed for the functioning of the blind navigation system.

DESCRIPTION

This class represents the LiDAR sensor.

This class manages TFMini LiDAR sensor and couples the microcontroller such that the executable only needs. to implement the data interface callback method. It runs a thread and uses UART for serial communication.
